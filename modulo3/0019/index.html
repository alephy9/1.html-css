Colocando imagens de fundo

Use no css ' background-image: url();

dentro de url, coloque aspas '' e digite o caminho do arquivo OU "Ctrl+Espaço"

exemplo: background-imagem: url('imagens/wallpaper.png');

Também pode colocar simplesmente uma url da web de uma imagem dentro de url('');

Imagens que se repetem

Use no css background-size: 'px 'px; pra configurar o tamanho

background-repeat: repeat; (padrão) serve pra repitir
background-repeat: no-repeat; aparece apenas UMA
background-repeat: repeat-x; repete horizontalmente
background-repeat: repeat-y; repete verticalmente
