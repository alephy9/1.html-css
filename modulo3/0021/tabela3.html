Alinhamento Vertical

Use vertical-align: baseline; (PADRÃO)

Todos elementos > baseline, length, sub, super, top, text-top, middle, bottom, text-bottom, initial, inheit

Também use: text-align, height, width

Escopo

Ajuda muito ao mecanismo de busca do Google, SEO.

< th scope= 'col' > [ Define que o cabeçalho pertence a linha vertical de < td > ]

< th scope= 'row' > [ Define que o cabeçalho pertence a linha horizontal de < td > ]

Efeito zebrado

Use uma pseudo-class ' :nth-child(2n) ' [ 2n define cor sim, cor não ]
(odd) define apenas ímpares, (even) define apenas pares
Sempre use a tag filho " > "

exemplo: tbody > tr :nth-child (2n) { ............................. }

Mesclagens de células

Use colspan= "2" (Ocupa 2 coluna na HORIZONTAL)

Use rowspan= "3" (Ocupa 3 coluna na VERTICAL)

exemplo:
< td colspan= "2" >
< td rowspan= "3" >
