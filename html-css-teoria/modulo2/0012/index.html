CSS estilos

Esse é o estilo que o Css dar em um site, primeiro capitulo da aula de CSS.

Existem 3 formas de utilizar

Estilo inline

Não é recomendado usar essa técnica, porque consiste em poluir os códigos porque enche o VScode de palavras.
Colocando o style nas tags html

Estilo interno

Essa técnica você pode configurar todos os códigos de um arquivo específico. É recomendado usar quando uma página web requer uma configuração específica diferente das outras.

Estilo externo

Essa é a melhor forma de utilizar o estilo css, você cria um 'link:css' abaixo de <title>,
daí então você pressiona 'Ctrl+clique' em 'style.css' e cria um arquivo específico que vai atender todas as páginas de arquivos
que contenham o 'link:css' desta pasta que foi criada.
