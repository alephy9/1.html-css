Formatações de Textos!

Semântica no HTML5 é importante, quer dizer que o significado é mais valioso do que a forma.
Pesquisar sobre as tags obsoletas no w3.org que mostra as que não funcionam mais, por exemplo as que ficam vermelhas no VScode.
Focar no significado das palavras, a forma deixa que o CSS3 faz.

Negrito e Itálico

Essa é a forma certa de deixar as letras em negrito!
Tag: <strong> </strong>


Essa é a forma certa de deixar as letras em itálico!
Tag: <em> </em>

Observação: Pra envelopar as mensagens pra inserir códigos rápidos no meio de palavras,
Pressione 'Ctrl+Shift+P' procure 'wrap' (wrap with abbreviation),
digite a tag que você quer colocar.

Texto Marcado

Esse é o método de como marcar um texto de forma correta.
<mark> </mark>
Observação: Pra mudar a cor do marca texto somente com o CSS3

Texto grande e pequeno

Essa tag de deixar grande está obsoleta.
Tag: <big> </big>

Essa tag de deixar pequeno está funcionando perfeitamente.
Tag: <small> </small>

Texto deletado e inserido

Essa é o jeito certo de deletar um texto, deixar riscado.
Tag: <del> </del>

Esse é o jeito certo de inserir um texto, deixar sublinhado.
Tag: <ins> </ins>

Texto sobrescrito e subscrito

Deixando sobrescrito: x3+x2+109
Tag: <sup> </sup>

Deixando subscrito: H20 , CO2
Tag: <sub> </sub>

Código-fonte

Serve pra fazer com que palavras sejam mostradas como formato de códigos.
Tag: <code> </code>

Pre-formatação

Serve pra organizar os códigos, mostrar no site da mesma forma como está escrito no VScode.
Tag: <pre> </pre>

Exemplo: Código Python 

        nome = "Nora"
        linguagem_favorita = "Python"
        
        print(f"Olá, eu sou {nome}. Estou aprendendo {linguagem_favorita}.")

Citações Simples

Isso mostra aspas simples, significa uma citação.
Tag: <q> </q>
Observação: sempre que precisar colocar aspas em uma citação simples, coloque o código!

Citações Completas

São citações longas, extensas... de autores e etc.
Tag: <blockquote> </blockquote>

Caso queira mostrar para o mecanismo de busca do Google de onde você tirou a citação.
Acrescente 'cite= "link do site" '
Tag: <blockquote cite="link do site"> </blockquote>

Exemplo:

O ser humano vivencia a si mesmo, seus pensamentos como algo separado do resto do universo – numa espécie de ilusão de ótica de sua consciência. E essa ilusão é uma espécie de prisão que nos restringe a nossos desejos pessoais, conceitos e ao afeto por pessoas mais próximas. Nossa principal tarefa é a de nos livrarmos dessa prisão, ampliando o nosso círculo de compaixão, para que ele abranja todos os seres vivos e toda a natureza em sua beleza. Ninguém conseguirá alcançar completamente esse objetivo, mas lutar pela sua realização já é por si só parte de nossa liberação e o alicerce de nossa segurança interior.
