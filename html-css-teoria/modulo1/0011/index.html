Imagens Dinâmicas

Serve para abrir esse site em outros aparelhos, ou diminuir o tamanho do seu navegador.


Criar a tag <picture> </picture>
Sempre fazer da foto grande p/ menor
dentro dessa tag, criar tag <img src='Ctrl+Espaço' alt='titulo' (colocar imagem grande)
acima da tag 'img' criar os <source:media:types>
<source media='min-width:digite quantidade pixel' srcset='digite local arquivo' type='image/digite o formato da foto'>
trocar media=min por max, afinal é do maior pro menor

segunda e mais recomendada é a forma que vai testar vários formatos e atender mais navegadores.
