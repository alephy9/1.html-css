Links Externos

Significa que você vai disponibilizar caminhos para links aos usuários no site, com links de outros sites já pronto!

Exemplo: Caminhando o usuário a um site externo, saindo do site de origem.
<a href= "cole link" > "Texto ou frase pro usuário clicar" </a>
Clique aqui.

Exemplo: Caminhando o usuário a um site externo, mas sem fechar o site de origem.
<a href="cole link" target="_blank" rel="external"> "Texto ou frase" </a>
_blank significa que o usuário vai abrir link em uma nova guia.
rel=external significa apenas pro mecanismo de pesquisa que é um site externo.
Clique aqui.

Exemplo: Caminhando o usuário a um site externo,
saindo do site de origem mas dando possibilidade de voltar facilmente na opção do navegador.
<a href='link' target="_self" rel="external"
_self significa que o usuário pode voltar ao site clicando na opção de "retornar" do navegador.
Clique aqui.

Observação: _blank ou _self, é algo bem opcional que vai de acordo com a facilidade de como vai ficar para o usuário.

Links Internos

Significa que você vai disponibilizar o site que você criou com códigos!

Exemplo: Caminhando o usuário a uma segunda e terceira página criada pelo Desenvolvedor.
<a href='Ctrl+Espaço' target="_blank" rel="next">"Texto.Frase"</a>
<a href='Ctrl+Espaço' target="_blank" rel="next">"Texto.Frase"</a>
(Ctrl+Espaço) pra encontrar o arquivo feito no pc, é bom que esteja na pasta que está aberta no VScode, fica mais rápido.
(rel="next") Como não é um site externo, nós indicamos ao mecanismo de pesquisa que é uma página adiante do site criado.

Links Patrocinados

Para links patrocinados, é bem simples.
<a href="Link.Site" target="_Blank" rel="nofollow">"Texto.Frase"</a>
Apenas copiar e colar o link do site
(rel="_nofollow") significa que seu site vai apresentar o link patrocinado normalmente,
mas pro mecanismo de pesquisa você não dar aval pra esse link.
Exemplo: Link Patrocinado.

Links Download

Você vai disponibilizar um arquivo no pc, para o usuário baixar através de um link.

<a href="Ctrl+Espaço" download="arquivo.formato" type="application/formato">"Texto.Frase"</a>
href= procure o arquivo na pasta que está aberta no VScode
download=apenas o nome do arquivo + formato (descarte nome pasta caso tenha)
type= media type para formato do arquivo (iana.org/assigments tem todos media types)
