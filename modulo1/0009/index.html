Listas Ordenadas

Essa tag serve para dar um ponta-pé inicial em criar uma lista de forma ordenada.
<ol> </ol>
Dentro de 'ol' inserimos <li> </li> para criar os tópicos
Exemplo: Tag 'ol' padrão.

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Agora saindo do padrão e configurando os tipos de listas ordenadas que existem,
Listas: numéricas, letras maiúsculas e minúsculas, números romanos grandes e pequenos.
Pra configurar é simples, basta adicionar 'type="digitar a forma"'

Exemplo: Lista numérica (Padrão)
<ol type=1> </ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Exemplo: Letras Maiúsculas
<ol type=A> </ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Exemplo: Letras minúsculas
<ol type=a> <ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Exemplo: Numéros Romanos Grandes
<ol type=I> <ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Exemplo: Números Romanos Pequenos
<ol type=i> <ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Pra configurar de onde vai começar a lista basta simplesmente adicionar 'start="indique sempre de forma numérica"
Exemplo: <ol type=1 start=5> </ol>

  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva
Exemplo: <ol type=A start=3> </ol>
  1. Francisco
  2. Alephy
  3. Nascimento
  4. Silva

Listas NÃO Ordenadas

Essa tag serve para dar um ponta-pé inicial em criar uma lista de forma NÃO ordenada.
<ul> </ul>
Dentro de 'ul' inserimos <li> </li> para criar os tópicos
Essa tag não é necessário adicionar 'start=' pois não tem ordem!

Exemplo: Tag 'ul' padrão
<ul> </ul>

  • Francisco
  • Alephy
  • Nascimento
  • Silva

Podemos configurar a tag da mesma forma adicionando 'type= disc, circle ou square'

Exemplo: Tag <ul type=circle> </ul>

  • Francisco
  • Alephy
  • Nascimento
  • Silva

Exemplo: Tag <ul type=square> </ul>

  • Francisco
  • Alephy
  • Nascimento
  • Silva

Listas Mescladas

É possível misturar as listas 'ul e ol' lembrando sempre de colocar 'li' depois que essas tags forem mencionadas,
e configurar de acordo com a mesma linha de raciocínio das tag's anteriores.

Exemplo: 'ol' 'li + (ul+li) + (ol+li)'
