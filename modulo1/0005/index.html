Parágrafos e quebra de linhas

Este parágrafo foi feito com <p> </p>

Este parágrafo, quebrou a linha usando
<br>, que não tem tag de fechamento.
Serve pra quebrar linhas dentro de parágrafos.
Observação: Jamais fazer várias tags 'br' com foco em dar espaço no site, isso a gente configura no CSS3.

Símbolos ®© e Emoji 🤑

Sobre símbolos e os emojis, precisa ter uma lista inteira de cola
pra poder ser mais rápido de achar e ver qual seria o código de determinado símbolo ou emoji para inserir. 🤕

Pra inserir emoji é simples, precisa digitar
(& # x ) + (Copiar código do Emoji a patir do '1f...') terminando com 'ponto e vírgula' ;
🤐 🤑 🤒 🤓 🤔

Pra inserir símbolos é simples, precisa digitiar apenas
( & ) + (nome do símbolo) terminando com 'ponto e vírgula' ;
® © € $ ǵ γ 𝔷

Observação: essa tag <!----> serve pra deixar algum comentário no VScode.
E essa é a tag que faz com que mostre os sinais ('& lt;' Menor que < ) e ('& gt;' Maior que >)
